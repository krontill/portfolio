Создание приложения на react&redux для работы с файлами.

Использовался дашборд Material Dashboard React на основе Material-UI с небольшими дизайнерскими изменениями.

Серверная часть на laravel, общение происходит с помощью REST API.

По функционалу была сделана работа с пользователями - авторизация, регистрация, восстановление пароля, страница профиля.

Есть обычные пользователи и есть админы и суперадмины, которые могут добавить, отредактировать или удалить пользователя.

Файлы разбиты по категориям.

Файлы можно добавлять, удалять, редактировать, делать скрытыми, просматривать превью и скачивать.

Админам также доступна статистика.
