Создание прототипа приложения на react&redux для отслеживания поставок лекарств.

Были созданы страница поиска с фильтрами и страница аналитики. Данные грузятся из файла csv.

При сворачивании фильтра видна карта. В боковой панеле показаны выбранные фильтры. Если кликнуть в таблице строку, то на карте отображается метка, которая показывает поставки из одного города в другой.

На каждой странице присутствует тур по приложению, объясняющий пользователю работу с данной страницей.
